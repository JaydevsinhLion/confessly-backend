# 💬 Confessly Backend

**Confessly** is an anonymous emotion-sharing app built with **Flask + MongoDB**, designed for pure, ephemeral expression.

---

## ⚙️ Stack
- Python 3.10+
- Flask 3.x
- MongoDB (Atlas)
- Flask-CORS
- TextBlob (Mood Analysis)
- JWT Auth for Admin Panel
